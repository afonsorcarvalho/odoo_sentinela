# Sentinela CME — Detalhe do Sensor

SPA em React + TypeScript + Vite que exibe o detalhe de um sensor (histórico,
limites regulatórios e cauda ao vivo em gráfico ECharts).

- Modo mock: defina `VITE_API_MODE=mock` (ver `src/lib/api`) para rodar sem backend.
- `npm run dev` — sobe o servidor de desenvolvimento.
- `npm test` — roda a suíte de testes (Vitest).
- `npm run build` — build de produção.
